# Dynamic Event Details Form

A front-end demo app for building ordered, structured invitation details with a live JSON preview.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- pnpm

## Setup

Install dependencies:

```bash
pnpm install
```

Run the local demo on port 3002:

```bash
pnpm dev
```

Open [http://localhost:3002](http://localhost:3002).

## Available Scripts

```bash
pnpm dev
pnpm build
pnpm start
```

## Notes

This is a front-end only demo. It uses local React state and does not include a backend, database, authentication, or API routes.
